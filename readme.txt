Клон qtscrob, написанный на J2SE (Swing + jCommons + last.fm_bindings). 
Поддерживается скробблинг прослушанного из текстового лога scrobbler.log; встроена защита от дублирования записей (повторные отметки о прослущивании, создаваемые при переключении треков или работе с плейлистом, игнорируются).
При отправке данных отображается статистика с графическим сопровождением.
